A propos de NewsDiffs

Pourquoi NewsDiffs Existe

De nos jour, le journalisme est digital, et rapidement changeant. Il y a rarement une version "définitive" d'un article

NewsDiffs observe différentes versions des articles des journaux en lignes les plus importants.

Pour le meilleur ou pour le pire, les lecteurs peuvent dorénavant voir les "coulisses" de l'édition, qui à l'époque du papier n'étaient pas accessible.

NewDiffs est né lors du Knight Mozilla MIT hackathon le 17 Juin 2012. Le code du projet est libre (sur GitHub), ce qui a permis la naissance de NewsDiffs en français.

Quels types de changements ?

Les principaux journaux en ligne sont constamment édités.

La plupart du temps, ces changements sont mineurs, de petits ajustement (par exemple "11" en "onze")

Parfois, les changements consistent en l'ajout ou la suppression d'un paragraphe.

Parfois, l'article change parce que le sujet est au coeur d'une actualité rapidement changeante, comme la crise diplomatique entre Paris et La Paz à propos de Edward Snowden. L'article est enrichi avec le temps, au fur et a mesure que l'information évolue. Dans cet article à propos du coup d'Etat contre le president Mohamed Morsi en Egypte, le nombre de victime des affrontements est mis à jour d'une "dizaine" à une "vingtaine", puis encore une fois avec un bilan officiel de seize morts

De simple changements de language peuvent avoir des implication importante sur la connotation. Par exemple, sur NewsDiffs americain, l'élection egyptienne est qualifiée de "democratique" puis "compétitive."

Parfois, le point de vue change complètement une fois que des informations additionelles sont pourvues. C'est le cas de cet article qui a inspiré le projet : le sujet traite des arrestations lors des manifestations du mouvement Occupy Wall Street le 1er Octobre 2011. Deux versions, à vingt minutes d'écart, voient leur premier paragraphe radicalement changé. Celui-ci traite Two versions, twenty minutes apart, had substantially different first paragraphs. Les différentes versions ne sont pas disponible au public, d'où l'intêret d'un project tel que NewsDiffs

D'où Vient le Nom NewsDiffs?
diff est un outil information qui permet d'afficher la différence entre deux fichiers. Il est habituellement utilisé pour comparer deux versions d'un même fichier. Le principe du contrôle de version est bien ancré dans le domaine du developpement logiciel, mais il devrais l'être aussi dans d'autres diciplines comme le journalisme.

Comment NewDiffs Fonctionne

NewsDiffs archive plusieurs fois par jour les articles à la Une des principaux journaux en ligne. Il extrait le contenu de ces articles et les stockent dans un repository git.

Tous les articles ne sont pas archivés. Seulement ceux qui ont des changement significatifs sont affichés.

Quels Outils Avez-Vous Utilisé ?

Le projet Newsdiffs original est libre, et son code source est disponible sur Github.

NewsDiffs en français est un fork du projet mentionné ci-dessus. Il est aussi libre, son code source est disponible sur Github.

La librairie Diff Match Patch est utilisée pour afficher les changements dans un article.

Le site utilise le framework Django.

L'apparence du site est due à Twitter Bootstrap
